# 增添sql.php

### 2. 前台的total顯示格式

增加以下代碼做提取。同時判別如果沒SESSION紀錄，先update值+1。之後再做網頁要顯示的select

```php
//t7
if(empty($_SESSION['visit'])){
    $_SESSION['visit']="welcome";
    $result=select("t7_total",1);
    $post['once']=$result[0]['once']+1;
    update($post,"t7_total");
}
$result=select("t7_total",1);
$t7total=$result[0]['once'];
```
