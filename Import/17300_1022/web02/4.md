# 第4題. 主選單

這題是送分題，題目要求的動作在板型上都已完成。不需要處理。所有的選單都要有內容連接。所以這裡做一下URL對應

# 增添sql.php

### 1. 規劃選單連結內容

設計$content\_zone代碼會依檔案名稱而名

```php
//for t4
$content_zone=(empty($_GET['do']))?"main":$_GET['do'];
```

# 修改index.php

找到位置

```php
<div class=""></div>
```

改成

```php
<div class=""><?php include $content_zone.".php"?></div>
```
